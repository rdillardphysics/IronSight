This folder contains app icons used by Tauri bundling.

Replace `icon.png` with a high-resolution PNG (512x512) or platform-specific icons:
- macOS: provide `icon.icns` for best results
- Windows: provide `icon.ico`
- Linux: provide PNGs (256x256 or 512x512)

Current `icon.png` is a placeholder; replace it with your official app artwork before building release installers.
